CSS (Cascading Style Sheets) is a language used for describing the presentation of a document written in HTML or XML. It controls how HTML elements are displayed on a web page. This tutorial will teach you CSS from basic to advanced.

What You Will Learn

  • Basic syntax and structure of CSS
  • Applying CSS styles to HTML elements
  • Working with selectors and properties
  • Understanding CSS box model
  • Responsive design techniques

Prerequisites

Before starting this tutorial, you should have a basic understanding of HTML and how web pages work.

Getting Started

Embedding CSS

CSS can be applied to HTML documents in three ways:

  1. Inline CSS: Applied directly within an HTML tag using the style attribute. 
    <p style="color: blue; font-size: 16px;">Hello, CSS!</p>
  2. Internal CSS: Defined within the <style> element in the <head> section of an HTML page<head>
        
    <head>
    <style>
        p {
            color: blue;
            font-size: 16px;
             background:red;

        }
    </style>
</head>
  3. External CSS: Defined in a separate CSS file and linked to an HTML document using the <link> element. 
    <head>
    <link rel="stylesheet" type="text/css" href="styles.css">
</head>
